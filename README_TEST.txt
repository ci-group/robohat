Internal test programs, to test the Robohat

connect serial plug to the topboard of the Raspberry
startup the Raspberry

login:		robo
password:	test


you now will enter the console of the Raspberry
There are test utilities located in ~/bin. This path is already available on the console.


Servo test program.
------------------
The servo test program is for internal use to test the hardware
To start the servo test program type

servo

Just press enter for the test menu
